# MiniProject

Unity를 공부하면서 배운 것을 바로 만들어보는 공간입니다.

완성도 높은 게임을 만드는 레포가 아니라, 하나의 기술적 목표를 잡고 → 구현하고 → 뭘 배웠는지 기록하는 흐름으로 운영합니다. 각 미니프로젝트는 독립된 학습 단위이고, 장기적으로는 개인 프로젝트 Fatile (타일 기반 로그라이크)의 시스템 기반이 될 예정.

## Projects

### 01. BattlePrototype

CSV로 몬스터 데이터를 관리하고, 코루틴 재귀로 턴제 전투를 구현한 프로토타입.  
기획 데이터를 코드 수정 없이 게임에 반영하는 구조를 직접 만들어보고 싶었습니다..

→ [스크립트 보기](./BattlePrototype%20Scripts)


### 02. BattleSystem
캐릭터 선택 시스템과 상태 머신 기반 턴제 전투를 구현한 프로토타입.
C# 핵심 개념(생성자, out, ref, enum+switch, switch 표현식, when)을 실전 구조에 직접 녹여냈습니다.
